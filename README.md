# max6675-esp-idf
An example of how to use esp-idf to read a MAX6675 temperature sensor on esp32c6

## Motivation

branch from https://github.com/jordeam/max6675-esp-idf/tree/main

## Pins map:

| ESP32 | MAX6675 |
|-------|---------|
|    13 | SO      |
|     - | NC      |
|    12 | SCK     |
|    21 | CS      |


## Compatibilty

The code was tested only on ESP32c6 ( DevkitM-1)
